WhiteLagoon is a production-ready ASP.NET Core MVC web application built using Clean Architecture, simulating a real-world resort and villa booking system with secure online payments and an administrative dashboard.


Key Features

- Clean Architecture (Domain-driven structure)
- ASP.NET Core MVC (.NET 8)
- Entity Framework Core & SQL Server
- Stripe payment integration
- ASP.NET Core Identity (Authentication & Authorization)
- Role-based access (Admin / User)
- Admin dashboard with charts & analytics
- Repository Pattern & Unit of Work
- Secure configuration (no secrets committed)


Customer Features
- Browse available villas and amenities
- View villa details and availability
- Secure online booking using Stripe
- User registration and authentication

Admin Features
- Administrative dashboard with booking and revenue analytics
- Manage villas, villa numbers and amenities
- Booking management (check-in / check-out)
- CMS-style admin interface
- Role-based authorization
